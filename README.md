#  Lista de Tarefas

App simples de lista de tarefas com:
-  Adicionar/remover tarefas
-  Marcar como concluídas
-  Filtrar por status
-  Salva automaticamente
-  Responsivo

## Tecnologias
- HTML5
- CSS3  
- JavaScript

## Como usar
1. Clone o repositório
2. Abra o index.html no navegador
